# Gif-Explorer-App
Gifs explorer app made with React and Giphy API

## Table of contents
* [Technologies](#technologies)
* [Setup](#setup)

## Technologies
Project is created with:
* JavaScript (ES6)
* HTML5
* CSS3
* React
* Vite

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm run dev
```
